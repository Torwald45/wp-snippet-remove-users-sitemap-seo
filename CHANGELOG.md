# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2025-10-15

### Added
- Remove users sitemap (wp-sitemap-users-1.xml) from WordPress Core Sitemaps
- Filter hook for wp_sitemaps_add_provider
- Optional commented code to remove taxonomies sitemap
- Optional commented code to remove posts sitemap
- Privacy protection by hiding author usernames
- Lightweight implementation (single filter)
- Support for WordPress 5.5+ Core Sitemaps
